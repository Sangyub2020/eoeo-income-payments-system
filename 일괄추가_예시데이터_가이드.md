## 💡 주요 포인트

### 1. 금액 형식 (다양한 형식 지원)
- **원화**: `₩5,000,000`, `5000000`, `5,000,000원`, `500만원`
- **달러**: `$10,000`, `10000`, `10,000 USD`, `USD 10000`
- 시스템이 자동으로 통화를 감지합니다.

### 2. 브랜드명 여러 개 입력 시
다음과 같은 형식이 모두 지원됩니다:
- **쉼표로 구분**: `COLORGRAM, BRING GREEN, WAKE MAKE`
- **쉼표+공백**: `COLORGRAM,BRING GREEN,WAKE MAKE`
- **쉼표+공백 혼합**: `COLORGRAM,BRING GREEN, WAKE MAKE`

### 3. 날짜 형식
- **YYYY-MM-DD 형식**: `2025-12-31`, `2025-01-15`
- 빈칸도 가능 (입금일이 아직 없을 경우)

### 4. 귀속연월 형식
- **YYMM 형식**: `2512` (2025년 12월), `2501` (2025년 1월)

### 5. 구분 필드 (필수)
다음 중 하나를 선택해야 합니다:
- `ONE-TIME`
- `B2B`
- `파트너십/마케팅지원비`
- `기재고사입`
- `정부지원사업`
- `배송비`
- `기재고판매`
- `other`

### 6. 선/잔금 필드
- `일시불`: 한 번에 납부
- `선금`: 선금 납부
- `잔금`: 잔금 납부

### 7. 세금계산서첨부상태
- `required`: 첨부필요
- `completed`: 첨부완료
- `not_required`: 첨부불요

### 8. 입금여부
- `입금완료`: 입금이 완료된 경우
- `입금예정`: 입금 예정인 경우
- `입금지연`: 입금이 지연된 경우

## 📝 사용 방법

1. **구글 시트에서 데이터 준비**
   - 위의 필드 순서에 맞춰 데이터를 입력합니다.
   - 각 행은 하나의 레코드를 나타냅니다.

2. **데이터 복사**
   - 구글 시트에서 데이터 영역을 선택합니다.
   - Ctrl+C (또는 Cmd+C)로 복사합니다.

3. **일괄추가 모달에 붙여넣기**
   - 대시보드에서 "일괄 추가" 버튼을 클릭합니다.
   - CSV 데이터 붙여넣기 영역에 복사한 데이터를 붙여넣습니다.
   - "파싱 및 확인" 버튼을 클릭합니다.

4. **데이터 확인 및 수정**
   - 파싱된 데이터를 확인합니다.
   - 필요한 경우 각 항목을 수정할 수 있습니다.
   - 브랜드명은 드롭다운에서 선택할 수 있습니다.

5. **일괄 등록**
   - 모든 데이터가 정확한지 확인한 후 "일괄 등록" 버튼을 클릭합니다.

## ⚠️ 주의사항

1. **탭 구분자**: 구글 시트에서 복사하면 탭(`\t`)으로 구분됩니다. 시스템이 자동으로 감지합니다.

2. **빈 칸 처리**: 빈 칸은 그대로 두면 됩니다. 시스템이 자동으로 NULL로 처리합니다.

3. **특수문자**: 따옴표(`"`)가 포함된 경우 자동으로 처리됩니다.

4. **금액 형식**: 다양한 형식을 지원하지만, 통화 기호가 명확하지 않으면 기본값(KRW)이 적용됩니다.

5. **브랜드명**: 쉼표로 구분된 여러 브랜드명은 자동으로 배열로 변환됩니다.
